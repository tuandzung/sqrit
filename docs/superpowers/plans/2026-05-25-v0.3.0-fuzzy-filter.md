# v0.3.0 — Fuzzy filter Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the v0.2 case-insensitive substring filter in Results with a fuzzy-subsequence filter (`nucleo-matcher`) that highlights matched characters inline. Ship as v0.3.0.

**Architecture:** New `src/filter.rs` owns a long-lived `nucleo_matcher::Matcher` and a `rank()` API that scores rows column-by-column. `ResultsState` stores the live query + cached hits. Rendering consults hits to apply `Modifier::BOLD | Modifier::UNDERLINED` + accent color to matched char ranges. PgDn re-runs `rank()` against each newly fetched page. V6 holds — never the full result set in memory.

**Tech Stack:** Rust 2021, ratatui 0.29, crossterm 0.28, tokio (existing). New dep: `nucleo-matcher = "0.3"` (BSD-2, pure Rust).

```rust
//! Fuzzy filter for Results rows.
//!
//! Replaces the v0.2 case-insensitive substring filter with `nucleo-matcher`'s
//! subsequence scorer. Operates on the currently loaded page only — V6 holds.

use std::ops::Range;

use nucleo_matcher::pattern::{CaseMatching, Normalization, Pattern};
use nucleo_matcher::{Config, Matcher, Utf32Str};

use crate::db::types::{QueryResult, ResultColumn, Row};

/// Cached matcher reused across keystrokes so the scratch buffers inside
/// `nucleo_matcher::Matcher` aren't reallocated per character.
pub struct FuzzyFilter {
    matcher: Matcher,
    haystack_buf: Vec<char>,
    indices: Vec<u32>,
}

impl Default for FuzzyFilter {
    fn default() -> Self {
        Self::new()
    }
}

impl FuzzyFilter {
    pub fn new() -> Self {
        Self {
            matcher: Matcher::new(Config::DEFAULT),
            haystack_buf: Vec::new(),
            indices: Vec::new(),
        }
    }

    /// Score every row in `result` against `query`. Returns hits ordered by
    /// descending score; ties broken by original row index for stability.
    ///
    /// An empty `query` returns one `FilterHit` per row in original order with
    /// `score = 0` and `matches = []`.
    pub fn rank(&mut self, result: &QueryResult, query: &str) -> Vec<FilterHit> {
        if query.trim().is_empty() {
            return (0..result.rows.len())
                .map(|row_index| FilterHit {
                    row_index,
                    score: 0,
                    matches: Vec::new(),
                })
                .collect();
        }
        let pattern = Pattern::parse(query.trim(), CaseMatching::Ignore, Normalization::Smart);
        let mut hits = Vec::new();
        for (row_index, row) in result.rows.iter().enumerate() {
            if let Some(hit) = self.score_row(row_index, row, &result.columns, &pattern) {
                hits.push(hit);
            }
        }
        hits.sort_by(|a, b| b.score.cmp(&a.score).then(a.row_index.cmp(&b.row_index)));
        hits
    }

    fn score_row(
        &mut self,
        row_index: usize,
        row: &Row,
        columns: &[ResultColumn],
        pattern: &Pattern,
    ) -> Option<FilterHit> {
        let mut total: u32 = 0;
        let mut matches: Vec<(usize, Range<usize>)> = Vec::new();
        for (col_idx, col) in columns.iter().enumerate() {
            let Some(value) = row.get(&col.name) else {
                continue;
            };
            let text = value.to_string();
            self.haystack_buf.clear();
            let haystack = Utf32Str::new(&text, &mut self.haystack_buf);
            self.indices.clear();
            if let Some(score) = pattern.indices(haystack, &mut self.matcher, &mut self.indices) {
                if score == 0 {
                    continue;
                }
                total = total.saturating_add(score as u32);
                for range in contiguous_ranges(&self.indices) {
                    matches.push((col_idx, range));
                }
            }
        }
        if total == 0 {
            return None;
        }
        Some(FilterHit {
            row_index,
            score: total.min(u16::MAX as u32) as u16,
            matches,
        })
    }
}

/// One scored row. `matches` is `(column index, char range)`; the range is in
/// `chars()` units of the column's `Display` rendering — the renderer must
/// walk by chars, not bytes.
#[derive(Debug, Clone)]
pub struct FilterHit {
    pub row_index: usize,
    pub score: u16,
    pub matches: Vec<(usize, Range<usize>)>,
}

/// Collapse a sorted vector of single-char positions into contiguous ranges.
/// e.g. `[0,1,2,5,6]` → `[0..3, 5..7]`.
fn contiguous_ranges(positions: &[u32]) -> Vec<Range<usize>> {
    let mut out = Vec::new();
    let mut iter = positions.iter().copied();
    let Some(first) = iter.next() else {
        return out;
    };
    let mut start = first as usize;
    let mut prev = first;
    for p in iter {
        if p == prev + 1 {
            prev = p;
        } else {
            out.push(start..(prev as usize + 1));
            start = p as usize;
            prev = p;
        }
    }
    out.push(start..(prev as usize + 1));
    out
}
```

```rust
//! Row → ratatui Spans with fuzzy-match highlight overlays.
//!
//! Extracted from `app.rs` so the highlight logic can be unit-tested without
//! standing up a full Frame/Terminal.

use ratatui::style::{Modifier, Style};
use ratatui::text::Span;

use crate::filter::FilterHit;
use crate::theme::Theme;

/// Build the inline `Vec<Span>` for one cell. `hit` may be `None` (no filter
/// or row not in hits). When `Some`, `matched_ranges` carries the char ranges
/// that should be highlighted for THIS column index.
pub fn render_cell(
    text: &str,
    matched_ranges: &[std::ops::Range<usize>],
    theme: &Theme,
) -> Vec<Span<'static>> {
    if matched_ranges.is_empty() {
        return vec![Span::raw(text.to_string())];
    }
    let chars: Vec<char> = text.chars().collect();
    let highlight = Style::default()
        .fg(theme.border_focused) // accent
        .add_modifier(Modifier::BOLD | Modifier::UNDERLINED);
    let mut spans = Vec::new();
    let mut cursor = 0usize;
    let mut ranges: Vec<std::ops::Range<usize>> = matched_ranges.to_vec();
    ranges.sort_by_key(|r| r.start);
    for range in ranges {
        if range.start >= chars.len() {
            break;
        }
        if range.start > cursor {
            spans.push(Span::raw(chars[cursor..range.start].iter().collect::<String>()));
        }
        let end = range.end.min(chars.len());
        spans.push(Span::styled(
            chars[range.start..end].iter().collect::<String>(),
            highlight,
        ));
        cursor = end;
    }
    if cursor < chars.len() {
        spans.push(Span::raw(chars[cursor..].iter().collect::<String>()));
    }
    spans
}

/// Pick out the matched ranges for `(row_index, col_index)` from a slice of
/// hits, in stable order.
pub fn matched_ranges_for<'a>(
    hits: &'a [FilterHit],
    row_index: usize,
    col_index: usize,
) -> impl Iterator<Item = &'a std::ops::Range<usize>> + 'a {
    hits.iter()
        .filter(move |h| h.row_index == row_index)
        .flat_map(move |h| {
            h.matches
                .iter()
                .filter_map(move |(c, r)| if *c == col_index { Some(r) } else { None })
        })
}

```
