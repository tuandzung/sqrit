# v0.4.2 Context-Aware Autocomplete Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Resolve aliases and namespace-qualified identifiers in Query Insert autocomplete, show candidate details, and insert only the required backend quoting without weakening the existing 300 ms bare-completion flow.

**Architecture:** Keep the v0.4.0 statement scanner as the only owner of editor ranges. Convert autocomplete to structured candidates, parse only the current statement with the active `sqlparser` dialect, and walk only top-level physical `FROM`/`JOIN` relations. Treat AST resolution as advisory: qualified completion returns no candidates on scan or parse failure, while bare keyword/type/schema completion bypasses the parser.

**Tech Stack:** Rust 2021, ratatui 0.29, existing SQLite/PostgreSQL/MySQL schema model, and `sqlparser` 0.62.

## Global Constraints

- Start from completed v0.4.1 and ship as `0.4.2`.
- Preserve V1-V11, especially the 300 ms idle trigger in V7.
- Use `sqlparser` only for top-level relation and alias resolution; do not replace the lossless scanner or syntax tokenizer.
- Support explicit and implicit aliases, top-level joins, namespaces, and `namespace.relation.column` paths.
- Resolve only tables, views, and materialized views from `SchemaInfo`.
- Let an alias or unaliased relation binding shadow a namespace with the same name.
- Keep CTEs, derived tables, correlated subqueries, nested scopes, clause-aware ranking, routines, fuzzy ranking, and MRU ranking unresolved.
- Quote only identifiers that require quoting for the active backend.
- When a qualifier is already in the editor, insert only the final segment.
- Keep config, history, and theme schemas unchanged; add no mode or invariant.

```rust
use sqlparser::ast::{ObjectNamePart, SetExpr, Statement, TableFactor};
use sqlparser::dialect::{Dialect, MySqlDialect, PostgreSqlDialect, SQLiteDialect};
use sqlparser::keywords::{
    Keyword, ALL_KEYWORDS, ALL_KEYWORDS_INDEX, RESERVED_FOR_COLUMN_ALIAS,
    RESERVED_FOR_TABLE_ALIAS, RESERVED_FOR_TABLE_FACTOR,
};
use sqlparser::parser::Parser;

use crate::config::DbType;

pub fn quote_identifier_if_needed(ident: &str, backend: &DbType) -> String {
    let mut chars = ident.chars();
    let safe_shape = chars
        .next()
        .is_some_and(|first| first == '_' || first.is_ascii_alphabetic())
        && chars.all(|ch| ch == '_' || ch.is_ascii_alphanumeric());
    let preserves_case = !matches!(backend, DbType::Postgres)
        || !ident.chars().any(|ch| ch.is_ascii_uppercase());
    let keyword = keyword_for(ident);
    let keyword_requires_quotes = keyword.is_some_and(|keyword| {
        dialect_reserves(keyword, backend)
            || RESERVED_FOR_COLUMN_ALIAS.contains(&keyword)
            || RESERVED_FOR_TABLE_ALIAS.contains(&keyword)
            || RESERVED_FOR_TABLE_FACTOR.contains(&keyword)
            || !parses_as_unquoted_identifier(ident, backend)
    });

    if safe_shape && preserves_case && !keyword_requires_quotes {
        return ident.to_string();
    }

    match backend {
        DbType::Sqlite => quote_sqlite(ident),
        DbType::Postgres => quote_pg(ident),
        DbType::Mysql => quote_mysql(ident),
    }
}

fn keyword_for(ident: &str) -> Option<Keyword> {
    ALL_KEYWORDS
        .iter()
        .zip(ALL_KEYWORDS_INDEX.iter())
        .find_map(|(spelling, keyword)| spelling.eq_ignore_ascii_case(ident).then_some(*keyword))
}

fn dialect_reserves(keyword: Keyword, backend: &DbType) -> bool {
    match backend {
        DbType::Sqlite => SQLiteDialect {}.is_reserved_for_identifier(keyword),
        DbType::Postgres => PostgreSqlDialect {}.is_reserved_for_identifier(keyword),
        DbType::Mysql => MySqlDialect {}.is_reserved_for_identifier(keyword),
    }
}

fn parses_as_unquoted_identifier(ident: &str, backend: &DbType) -> bool {
    let probe = format!("SELECT 1 FROM {ident}");
    let statements = match backend {
        DbType::Sqlite => Parser::parse_sql(&SQLiteDialect {}, &probe),
        DbType::Postgres => Parser::parse_sql(&PostgreSqlDialect {}, &probe),
        DbType::Mysql => Parser::parse_sql(&MySqlDialect {}, &probe),
    };
    let Ok(statements) = statements else {
        return false;
    };
    let [Statement::Query(query)] = statements.as_slice() else {
        return false;
    };
    let SetExpr::Select(select) = query.body.as_ref() else {
        return false;
    };
    let [table] = select.from.as_slice() else {
        return false;
    };
    if !table.joins.is_empty() {
        return false;
    }
    let TableFactor::Table {
        name,
        alias: None,
        args: None,
        ..
    } = &table.relation
    else {
        return false;
    };
    let [ObjectNamePart::Identifier(found)] = name.0.as_slice() else {
        return false;
    };
    found.quote_style.is_none() && found.value == ident
}
```

The parser probe runs only for known keyword spellings that already pass the ASCII shape check, so normal schema names do not pay for parsing and candidate text cannot become SQL structure. A keyword is quoted when the active dialect reserves it, sqlparser reserves it in an identifier-adjacent context, or the dialect cannot round-trip it as a plain table name. Keep the module-level documentation accurate by stating that the helper checks shape, PostgreSQL case folding, and dialect parsing.

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct CompletionCandidate {
    pub label: String,
    pub detail: Option<String>,
    pub insert_text: String,
}

#[derive(Default)]
pub struct AutocompleteState {
    candidates: Vec<CompletionCandidate>,
    filtered_indices: Vec<usize>,
    selected: usize,
    visible: bool,
    scope: String,
}

impl AutocompleteState {
    pub fn new() -> Self {
        Self::default()
    }

    pub fn open(&mut self, candidates: Vec<CompletionCandidate>, scope: String) {
        if candidates.is_empty() {
            self.dismiss();
            return;
        }
        self.candidates = candidates;
        self.filtered_indices = (0..self.candidates.len()).collect();
        self.selected = 0;
        self.visible = true;
        self.scope = scope;
    }

    pub fn is_visible(&self) -> bool {
        self.visible
    }

    pub fn filtered(&self) -> Vec<&CompletionCandidate> {
        self.filtered_indices
            .iter()
            .map(|&index| &self.candidates[index])
            .collect()
    }

    pub fn selected_index(&self) -> usize {
        self.selected
    }

    pub fn scope(&self) -> &str {
        &self.scope
    }

    pub fn dismiss(&mut self) {
        self.visible = false;
    }

    pub fn accept(&mut self) -> Option<CompletionCandidate> {
        if !self.visible {
            return None;
        }
        let choice = self
            .filtered_indices
            .get(self.selected)
            .map(|&index| self.candidates[index].clone());
        self.visible = false;
        choice
    }

    pub fn next(&mut self) {
        if !self.filtered_indices.is_empty() {
            self.selected = (self.selected + 1) % self.filtered_indices.len();
        }
    }

    pub fn prev(&mut self) {
        if !self.filtered_indices.is_empty() {
            self.selected =
                (self.selected + self.filtered_indices.len() - 1) % self.filtered_indices.len();
        }
    }

    pub fn filter(&mut self, prefix: &str) {
        self.filtered_indices = self
            .candidates
            .iter()
            .enumerate()
            .filter(|(_, candidate)| {
                candidate
                    .label
                    .get(..prefix.len())
                    .is_some_and(|head| head.eq_ignore_ascii_case(prefix))
            })
            .map(|(index, _)| index)
            .collect();
        self.selected = 0;
    }
}
```

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct CompletionInput {
    pub qualifiers: Vec<String>,
    pub prefix: String,
    pub replace_len: usize,
    pub scope: String,
}

pub fn completion_input(text: &str, row: usize, col: usize) -> Option<CompletionInput> {
    let line = text.split('\n').nth(row)?;
    if col > line.len() || !line.is_char_boundary(col) {
        return None;
    }

    let bytes = line.as_bytes();
    let mut prefix_start = col;
    while prefix_start > 0 && is_identifier_byte(bytes[prefix_start - 1]) {
        prefix_start -= 1;
    }
    if prefix_start > 0 && matches!(bytes[prefix_start - 1], b'\"' | b'`') {
        return None;
    }

    let prefix = line[prefix_start..col].to_string();
    let mut qualifiers = Vec::new();
    let mut position = prefix_start;
    while position > 0 && bytes[position - 1] == b'.' {
        position -= 1;
        let (start, value) = complete_segment_ending_at(line, position)?;
        qualifiers.push(value);
        position = start;
    }
    qualifiers.reverse();

    if qualifiers.is_empty() && prefix.is_empty() {
        return None;
    }
    let scope = if qualifiers.is_empty() {
        String::new()
    } else {
        format!("{}.", qualifiers.join("."))
    };
    Some(CompletionInput {
        qualifiers,
        prefix,
        replace_len: col - prefix_start,
        scope,
    })
}

fn is_identifier_byte(byte: u8) -> bool {
    byte.is_ascii_alphanumeric() || matches!(byte, b'_' | b'$')
}

fn complete_segment_ending_at(line: &str, end: usize) -> Option<(usize, String)> {
    if end == 0 {
        return None;
    }
    let bytes = line.as_bytes();
    let last = bytes[end - 1];
    if matches!(last, b'\"' | b'`') {
        let quote = last;
        let mut index = end - 1;
        while index > 0 {
            index -= 1;
            if bytes[index] != quote {
                continue;
            }
            if index > 0 && bytes[index - 1] == quote {
                index -= 1;
                continue;
            }
            let raw = &line[index + 1..end - 1];
            let doubled = if quote == b'\"' { "\"\"" } else { "``" };
            let single = if quote == b'\"' { "\"" } else { "`" };
            return Some((index, raw.replace(doubled, single)));
        }
        return None;
    }

    let mut start = end;
    while start > 0 && is_identifier_byte(bytes[start - 1]) {
        start -= 1;
    }
    (start < end).then(|| (start, line[start..end].to_string()))
}
```

```rust
use std::collections::{HashMap, HashSet};

use crate::config::DbType;
use crate::db::quote::quote_identifier_if_needed;
use crate::db::types::{ColumnInfo, SchemaInfo};

#[derive(Clone, Copy)]
struct SchemaRelation<'a> {
    namespace: &'a str,
    name: &'a str,
    columns: &'a [ColumnInfo],
}

fn schema_relations(schema: &SchemaInfo) -> Vec<SchemaRelation<'_>> {
    let mut relations = Vec::new();
    for namespace in &schema.namespaces {
        relations.extend(namespace.tables.iter().map(|table| SchemaRelation {
            namespace: &namespace.name,
            name: &table.name,
            columns: &table.columns,
        }));
        relations.extend(namespace.views.iter().map(|view| SchemaRelation {
            namespace: &namespace.name,
            name: &view.name,
            columns: &view.columns,
        }));
        relations.extend(
            namespace
                .materialized_views
                .iter()
                .map(|view| SchemaRelation {
                    namespace: &namespace.name,
                    name: &view.name,
                    columns: &view.columns,
                }),
        );
    }
    relations
}

fn prefix_matches(candidate: &str, prefix: &str) -> bool {
    candidate
        .get(..prefix.len())
        .is_some_and(|head| head.eq_ignore_ascii_case(prefix))
}

pub fn suggest_bare(
    prefix: &str,
    backend: &DbType,
    schema: Option<&SchemaInfo>,
) -> Vec<CompletionCandidate> {
    if prefix.is_empty() {
        return Vec::new();
    }

    let mut seen = HashSet::new();
    let mut candidates = Vec::new();
    for label in crate::sql::keywords()
        .iter()
        .chain(crate::sql::types().iter())
        .copied()
    {
        if prefix_matches(label, prefix) && seen.insert(label.to_ascii_lowercase()) {
            candidates.push(CompletionCandidate {
                label: label.to_string(),
                detail: None,
                insert_text: label.to_string(),
            });
        }
    }

    let Some(schema) = schema else {
        return candidates;
    };
    let relations = schema_relations(schema);
    let counts = relations.iter().fold(HashMap::new(), |mut counts, relation| {
        *counts
            .entry(relation.name.to_ascii_lowercase())
            .or_insert(0usize) += 1;
        counts
    });

    for relation in &relations {
        if !prefix_matches(relation.name, prefix) {
            continue;
        }
        let ambiguous = counts[&relation.name.to_ascii_lowercase()] > 1;
        let name = quote_identifier_if_needed(relation.name, backend);
        let insert_text = if ambiguous && !relation.namespace.is_empty() {
            format!(
                "{}.{}",
                quote_identifier_if_needed(relation.namespace, backend),
                name
            )
        } else {
            name
        };
        candidates.push(CompletionCandidate {
            label: relation.name.to_string(),
            detail: ambiguous.then(|| relation.namespace.to_string()),
            insert_text,
        });
    }

    for relation in relations {
        for column in relation.columns {
            let key = column.name.to_ascii_lowercase();
            if prefix_matches(&column.name, prefix) && seen.insert(key) {
                candidates.push(CompletionCandidate {
                    label: column.name.clone(),
                    detail: Some(column.data_type.clone()),
                    insert_text: quote_identifier_if_needed(&column.name, backend),
                });
            }
        }
    }
    candidates
}

pub fn suggest(prefix: &str, schema: Option<&SchemaInfo>) -> Vec<CompletionCandidate> {
    suggest_bare(prefix, &DbType::Sqlite, schema)
}
```

```rust
use sqlparser::ast::{ObjectName, ObjectNamePart, SetExpr, Statement, TableFactor};
use sqlparser::dialect::{MySqlDialect, PostgreSqlDialect, SQLiteDialect};
use sqlparser::parser::Parser;

use crate::sql::statement_at_cursor;

fn cursor_byte_offset(text: &str, cursor: (usize, usize)) -> Option<usize> {
    let (target_row, target_col) = cursor;
    let mut offset = 0usize;
    for (row, line) in text.split('\n').enumerate() {
        if row == target_row {
            return (target_col <= line.len() && line.is_char_boundary(target_col))
                .then_some(offset + target_col);
        }
        offset += line.len() + 1;
    }
    None
}

fn patched_current_statement(
    sql: &str,
    cursor: (usize, usize),
    backend: &DbType,
    input: &CompletionInput,
) -> Option<String> {
    let selected = statement_at_cursor(sql, cursor, backend.clone()).ok()??;
    let cursor_byte = cursor_byte_offset(sql, cursor)?;
    if cursor_byte < selected.range.start + input.replace_len
        || cursor_byte > selected.range.end
    {
        return None;
    }
    let local_cursor = cursor_byte - selected.range.start;
    let mut statement = sql[selected.range].to_string();
    statement.replace_range(
        local_cursor - input.replace_len..local_cursor,
        "__sqrit_completion",
    );
    Some(statement)
}

fn parse_one(sql: &str, backend: &DbType) -> Option<Statement> {
    let statements = match backend {
        DbType::Sqlite => Parser::parse_sql(&SQLiteDialect {}, sql),
        DbType::Postgres => Parser::parse_sql(&PostgreSqlDialect {}, sql),
        DbType::Mysql => Parser::parse_sql(&MySqlDialect {}, sql),
    }
    .ok()?;
    if statements.len() != 1 {
        return None;
    }
    statements.into_iter().next()
}
```

Append the top-level-only context implementation:

```rust
#[derive(Debug, Clone, PartialEq, Eq)]
struct RelationBinding {
    qualifier: String,
    namespace: Option<String>,
    relation: String,
}

#[derive(Debug, Clone, PartialEq, Eq)]
struct TopLevelContext {
    bindings: Vec<RelationBinding>,
    opaque_qualifiers: Vec<String>,
}

fn object_name_identifiers(name: &ObjectName) -> Option<Vec<String>> {
    name.0
        .iter()
        .map(|part| match part {
            ObjectNamePart::Identifier(identifier) => Some(identifier.value.clone()),
            _ => None,
        })
        .collect()
}

fn physical_binding(factor: &TableFactor) -> Option<RelationBinding> {
    let TableFactor::Table {
        name,
        alias,
        args: None,
        ..
    } = factor
    else {
        return None;
    };
    let parts = object_name_identifiers(name)?;
    let relation = parts.last()?.clone();
    let namespace = (parts.len() >= 2).then(|| parts[parts.len() - 2].clone());
    let qualifier = alias
        .as_ref()
        .map(|alias| alias.name.value.clone())
        .unwrap_or_else(|| relation.clone());
    Some(RelationBinding {
        qualifier,
        namespace,
        relation,
    })
}

fn opaque_qualifier(factor: &TableFactor) -> Option<String> {
    match factor {
        TableFactor::Table {
            name,
            alias,
            args: Some(_),
            ..
        } => alias
            .as_ref()
            .map(|alias| alias.name.value.clone())
            .or_else(|| object_name_identifiers(name)?.last().cloned()),
        TableFactor::Derived {
            alias: Some(alias), ..
        }
        | TableFactor::NestedJoin {
            alias: Some(alias), ..
        } => Some(alias.name.value.clone()),
        _ => None,
    }
}

fn collect_factor(context: &mut TopLevelContext, factor: &TableFactor) {
    if let Some(binding) = physical_binding(factor) {
        context.bindings.push(binding);
    } else if let Some(qualifier) = opaque_qualifier(factor) {
        context.opaque_qualifiers.push(qualifier);
    }
}

fn top_level_context(statement: &Statement) -> Option<TopLevelContext> {
    let Statement::Query(query) = statement else {
        return None;
    };
    if query.with.is_some() {
        return None;
    }
    let SetExpr::Select(select) = query.body.as_ref() else {
        return None;
    };

    let mut context = TopLevelContext {
        bindings: Vec::new(),
        opaque_qualifiers: Vec::new(),
    };
    for table in &select.from {
        collect_factor(&mut context, &table.relation);
        for join in &table.joins {
            collect_factor(&mut context, &join.relation);
        }
    }
    Some(context)
}

fn same_identifier(left: &str, right: &str) -> bool {
    left.eq_ignore_ascii_case(right)
}

fn resolve_binding<'a>(
    binding: &RelationBinding,
    schema: &'a SchemaInfo,
) -> Option<SchemaRelation<'a>> {
    let mut matches = schema_relations(schema).into_iter().filter(|relation| {
        same_identifier(relation.name, &binding.relation)
            && binding
                .namespace
                .as_deref()
                .map_or(true, |namespace| same_identifier(relation.namespace, namespace))
    });
    let first = matches.next()?;
    matches.next().is_none().then_some(first)
}
```

This never recurses into a derived table, nested join, table function, or nested query. Their top-level aliases are recorded only as opaque qualifiers so they shadow same-named namespaces without producing candidates. Any query with a `WITH` clause is rejected before collection.

```rust
fn column_candidates(
    relation: SchemaRelation<'_>,
    prefix: &str,
    backend: &DbType,
) -> Vec<CompletionCandidate> {
    relation
        .columns
        .iter()
        .filter(|column| prefix_matches(&column.name, prefix))
        .map(|column| CompletionCandidate {
            label: column.name.clone(),
            detail: Some(column.data_type.clone()),
            insert_text: quote_identifier_if_needed(&column.name, backend),
        })
        .collect()
}

fn namespace_candidates(
    namespace: &str,
    prefix: &str,
    backend: &DbType,
    schema: &SchemaInfo,
) -> Vec<CompletionCandidate> {
    schema_relations(schema)
        .into_iter()
        .filter(|relation| {
            same_identifier(relation.namespace, namespace)
                && prefix_matches(relation.name, prefix)
        })
        .map(|relation| CompletionCandidate {
            label: relation.name.to_string(),
            detail: None,
            insert_text: quote_identifier_if_needed(relation.name, backend),
        })
        .collect()
}

fn qualified_candidates(
    input: &CompletionInput,
    context: &TopLevelContext,
    backend: &DbType,
    schema: &SchemaInfo,
) -> Vec<CompletionCandidate> {
    let Some(first_qualifier) = input.qualifiers.first() else {
        return Vec::new();
    };
    if context
        .opaque_qualifiers
        .iter()
        .any(|qualifier| same_identifier(qualifier, first_qualifier))
    {
        return Vec::new();
    }
    let bindings = &context.bindings;
    match input.qualifiers.as_slice() {
        [qualifier] => {
            let matching_bindings = bindings
                .iter()
                .filter(|binding| same_identifier(&binding.qualifier, qualifier))
                .collect::<Vec<_>>();
            if matching_bindings.len() == 1 {
                return resolve_binding(matching_bindings[0], schema)
                    .map(|relation| column_candidates(relation, &input.prefix, backend))
                    .unwrap_or_default();
            }
            if !matching_bindings.is_empty() {
                return Vec::new();
            }
            namespace_candidates(qualifier, &input.prefix, backend, schema)
        }
        [namespace, relation_name] => {
            if bindings
                .iter()
                .any(|binding| same_identifier(&binding.qualifier, namespace))
            {
                return Vec::new();
            }
            schema_relations(schema)
                .into_iter()
                .find(|relation| {
                    same_identifier(relation.namespace, namespace)
                        && same_identifier(relation.name, relation_name)
                })
                .map(|relation| column_candidates(relation, &input.prefix, backend))
                .unwrap_or_default()
        }
        _ => Vec::new(),
    }
}

pub fn suggest_at_cursor(
    sql: &str,
    cursor: (usize, usize),
    backend: &DbType,
    schema: Option<&SchemaInfo>,
) -> Vec<CompletionCandidate> {
    let Some(input) = completion_input(sql, cursor.0, cursor.1) else {
        return Vec::new();
    };
    if input.qualifiers.is_empty() {
        return suggest_bare(&input.prefix, backend, schema);
    }

    let Some(schema) = schema else {
        return Vec::new();
    };
    let Some(statement_sql) = patched_current_statement(sql, cursor, backend, &input) else {
        return Vec::new();
    };
    let Some(statement) = parse_one(&statement_sql, backend) else {
        return Vec::new();
    };
    let Some(context) = top_level_context(&statement) else {
        return Vec::new();
    };
    qualified_candidates(&input, &context, backend, schema)
}
```

```markdown
After 300 ms idle in Query Insert, autocomplete produces structured candidates with a display label, optional detail, and insertion text. Bare completion suggests SQL keywords, types, physical relations, and columns. Duplicate bare relation names remain separate rows with namespace details; selecting an ambiguous relation inserts its qualified path.
```
